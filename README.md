## Abstract  
We present **QCMR-T v3.0**, a fully simulated, open-source platform coupling **superconducting transmon qubits** to **quantized vortices** in a rotating ³He-⁴He droplet at **0.9 mK**.  

**Key Results (Digital Twin: COMSOL + IBM Heron + Qiskit)**  
- Effective coupling: **Γ_eff = (3.8 ± 0.2) × 10⁻¹⁵ rad·m²**  
- Dephasing: **ΔT₂ = 0.38 ± 0.05 ns per vortex**  
- Braiding fidelity: **98.3% (4-vortex cycle)**  
- Bell violation: **S = 2.71 ± 0.03**
